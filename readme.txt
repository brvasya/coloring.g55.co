POOL PRODUCTION CHECKLIST FOR COLORING PAGE GENERATION

Character pool checklist
✔ Uses high demand characters or themes
✔ Covers popular and generic variants
✔ Names match common search queries
✔ No accidental duplicates unless weighting intentionally
✔ Characters compatible with all actions
✔ Characters suitable for kid friendly scenes
✔ No overly scene specific characters
✔ Pool large enough to avoid repetition
✔ Ambiguous names clarified for AI generation
✔ Avoid names AI may interpret as objects or food
✔ Add franchise or species clarification when needed

Action pool checklist
✔ No environments inside actions
✔ No plural forms if generator requires singular
✔ Actions work in any environment
✔ Actions fit all characters
✔ Printable friendly composition
✔ No complex motion requirements
✔ No violent or unsuitable actions

Environment pool checklist
✔ Printable friendly backgrounds
✔ Avoid crowd or city clutter
✔ Avoid environments requiring specific actions
✔ Reusable across characters and actions
✔ Avoid perspective heavy scenes
✔ Suitable for black and white line art

Compatibility checklist
✔ Every character works with every action
✔ Every action works in every environment
✔ No action requires specific environment
✔ No environment requires specific action
✔ Generator combinations remain natural

SEO checklist
✔ Pools include high demand search entities
✔ Generated titles sound natural
✔ Keywords match coloring page queries
✔ Category supports thousands of variations
✔ Evergreen themes preferred

Production safety checklist
✔ Pools allow 100k+ combinations
✔ No repeated scene patterns
✔ Safe for cron automation
✔ No generator failure scenarios
✔ No prompt contradictions

Scaling checklist
✔ Category expandable without redesign
✔ New characters easily added
✔ Actions reusable across themes
✔ Environments reusable across themes

Golden rule
Characters × Actions × Environments must always produce a valid printable coloring scene.
